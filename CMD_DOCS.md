# Документация по PassWDGenerator

## Основная команда

```bash
python main.py
```

Это откроет главное меню с опциями:
- 1. Сделать пароль
- 0. Выйти

При выборе "1" будет сгенерирован и отображен пароль длиной 16 символов.

> Примечание: При запуске терминал автоматически очищается перед отображением меню.
> 
> После выбора "1. Сделать пароль" пользователю предлагается выбрать:
> - "1" - сгенерировать один пароль
> - "много" - сгенерировать несколько паролей
> - "0" - вернуться в главное меню
> 
> После выбора "много" можно указать количество паролей (1-10)

## Команды

### generate
Генерирует пароли с возможностью настройки длины, количества и состава символов.

#### Параметры:
- `--length` или `-l`: Длина пароля (по умолчанию 16, минимум 4, максимум 128)
- `--count` или `-c`: Количество паролей для генерации (по умолчанию 1, минимум 1, максимум 10)
- `--include-digits` или `--no-digits`: Включать или исключать цифры (по умолчанию включены)
- `--include-punct` или `--no-punct`: Включать или исключать знаки препинания (по умолчанию включены)
- `--exclude-ambiguous` или `--include-ambiguous`: Исключать или включать неоднозначные символы (l, 1, I, O, 0) (по умолчанию не исключаются)

#### Примеры:
```bash
# Генерация одного пароля длиной 16 символов (по умолчанию)
python main.py generate

# Генерация одного пароля длиной 20 символов
python main.py generate --length 20

# Генерация 5 паролей длиной 16 символов
python main.py generate --count 5

# Генерация 3 паролей длиной 12 символов
python main.py generate --length 12 --count 3
```

### interactive
Интерактивный режим, в котором можно ввести параметры через консоль.

#### Пример:
```bash
python main.py interactive
```

Программа запросит ввод длины пароля и количества паролей для генерации.
